# 电商客服常见问题知识库


## 退换货政策
本店支持7天无理由退货。退货商品需保持完好，不影响二次销售，包装配件齐全。
以下商品不支持无理由退货：贴身衣物、已拆封的食品、定制商品、虚拟商品。
退货运费承担规则：
- 商品质量问题、发错货：卖家承担来回运费
- 7天无理由退货：买家承担退货运费
- 拒收/未签收：买家承担来回运费（质量问题除外）
- 换货运费：质量问题卖家承担，非质量问题买家承担
退货申请流程：
1. 进入订单详情页，点击"申请售后"
2. 选择"退货退款"，填写退货原因
3. 上传商品照片和相关凭证
4. 等待审核，通常1-3个工作日内完成
5. 审核通过后按系统提示地址寄回商品
6. 卖家收到货确认无误后，退款3-5个工作日内到账


## 退款流程
退款到账时间：
- 支付宝/微信支付：审核通过后3-5个工作日
- 银行卡支付：审核通过后5-7个工作日
- 花呗/白条：审核通过后1-3个工作日
退款进度查询：在订单详情页点击"退款详情"，可查看退款处理进度。
退款未到账怎么办：请先确认支付方式是否正确，确认退款状态是否为"已退款"。如超过上述时间仍未到账，请联系客服并提供订单号。


## 物流查询
如何查快递：登录账户进入"我的订单"，点击对应订单的"查看物流"按钮即可实时查看包裹位置和配送进度。
物流不更新怎么办：物流信息通常在发货后24小时内首次更新。如超过48小时无更新，请联系客服查询。
快递显示签收但我没收到：请先确认家人/同事/前台是否代收。如确实未收到，请在48小时内联系客服，超时可能无法处理。
配送范围：全国大部分地区配送（港澳台及偏远地区请咨询客服）。部分商品因属性限制不支持部分区域配送，下单时系统会自动提示。
配送时效：
- 普通快递：3-5个工作日
- 顺丰加急：1-2个工作日（需额外支付加急费）
- 大件商品：5-7个工作日，部分地区支持送货上门安装


## 售后保修
保修政策：电子产品质保1年，服装鞋包质保30天，食品类不支持质保。质保期内非人为损坏免费维修或换新。
申请保修流程：
1. 联系客服说明故障情况
2. 提供订单号与商品照片/视频
3. 客服评估是否需要寄回检修
4. 寄回检测后确定维修方案
5. 维修完成寄回或直接换新
以下情况不在保修范围：
- 人为损坏（摔坏、进水、自行拆修等）
- 超过质保期限
- 无法提供有效购买凭证
- 自然灾害等不可抗力造成的损坏
